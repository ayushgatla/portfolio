## 👋 About

Personal portfolio website for **Ayush Gatla**, a B.Tech CSE student at **IIITDM Kurnool** — a man who writes code, breaks it, Googles the error, and repeats. Built to showcase projects, skills, and a suspiciously good taste in Pokémon GIFs.

Features a dark/light mode toggle (because developers have opinions about their themes), smooth animations, and enough personality to make you scroll all the way to the contact section.

---

## ✨ Features

- 🌗 **Dark / Light Mode** — Toggle between moonlight and sunshine. Pick your villain arc.
- 📱 **Responsive Design** — Works on your phone, your laptop, and probably your fridge if it has a browser.
- 🎞️ **Animated Sections** — Custom GIFs and transitions so it doesn't feel like a 2009 resume.
- 🧠 **Projects Showcase** — Real work. Not tutorial clones. (Mostly.)
- 📬 **Contact Section** — Actually has social links, unlike most portfolios that just say "reach out!"

## ⚙️ Setup & Run Locally

Because apparently some people don't just open `index.html` and call it a day:

```bash
# Clone the repo
git clone https://github.com/ayushgatla/portfolio.git

# Navigate into it
cd portfolio

# Open in browser (it's just HTML, you're overthinking this)
open index.html
```

No npm install. No webpack config. No existential crisis. Just HTML.
